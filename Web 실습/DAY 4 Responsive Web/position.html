position : static

요소의 기본값으로, 요소들은 문서의 흐름에 따라 배치됨

position : relative

요소를 문서의 흐름에 따라 배치하고, top, bottom, left, right 값을 사용하여 위치를 이동할 수 있습니다.

position : absolute

요소를 문서의 원래 위치에서 제거하고, 가장 가까운 상위요소 중 relative를 찾아 해당 요소에 대해 상대적으로 배치합니다. 만약, 상위 요소 중 relative를 찾을 수 없다면 body tag를 기준으로 배치됩니다.

position : fixed

요소를 문서의 원래 위치에서 제거하고, 뷰포트를 기준으로 고정합니다.

position : sticky

요소를 문서에 흐름에 따라 배치하고, 스크롤을 통해 정해둔 임계점에 도달하면, fixed로 바뀐다.
